## Inside Tmux
### Prefix:
`Ctrl + Space`

### Windows
- New Window:
  `<prefix> + c`
- Jump to Window by index:
  `<prefix> + <window index>`
- Scroll through all open Windows
  `Ctrl + n`
  `Ctrl + p`

### Panes
- New Vertical Pane:
  `<prefix> + %`
- New Horizontal Pane:
  `<prefix> + "`
- Navigate through Panes in a Window
  `Ctrl + h/j/k/l`
